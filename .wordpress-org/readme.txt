== Description ==

An awesome tool to enable two step verification system on your WordPress site.

After a successful login, it sends a verification code to user's email/phone. And then the user needs to submit this code in the verification page to access admin area. Sounds good?

== Installation ==

1. Upload the `two-step-verification` folder and its contents to your `wp-contents/plugins` directory.
2. Activate in the `Plugins` menu.
3. Configure from WordPress admin area.
4. You are done!
